================================================================================
                         HOW TO RUN IT
================================================================================

STEP 1: Install Dependencies
   pip install -r requirements.txt

STEP 2: Configure Environment
   cp .env.example .env
   (Edit .env - add HF_TOKEN or GROQ_API_KEY, both free)

STEP 3: Initialize Database
   python init_db.py
   (Creates tables - already tested)

STEP 4: Ingest Sample Documents
   python ingest_sample.py
   (Creates 3 sample docs with embeddings)

STEP 5: Start Backend (Terminal 1)
   python -m uvicorn backend.main:app --reload
   (Starts at http://localhost:8000)

STEP 6: Start Frontend (Terminal 2)
   cd frontend && npm install && npm run dev
   (Starts at http://localhost:5173)

STEP 7: Test It!
   Option A: http://localhost:8000/docs (Swagger UI)
   Option B: http://localhost:5173 (React Frontend)
   Option C: curl request to http://localhost:8000/api/search

================================================================================
                         TROUBLESHOOTING QUICK REFERENCE
================================================================================

"Cannot connect to PostgreSQL"
   Check: pg_isready -h localhost -p 5432
   Start: services.msc -> PostgreSQL -> Start

"Port 8000 already in use"
   Use: python -m uvicorn backend.main:app --port 8001

"Module not found"
   Run: pip install -r requirements.txt

"Database does not exist"
   Run: python init_db.py

"HuggingFace 401 error"
   Solution: Set HF_TOKEN in .env file

"Embedding timeout"
   Note: First embedding takes 30-60s (model loading)
   Subsequent embeddings are much faster
